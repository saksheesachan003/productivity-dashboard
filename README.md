# Productivity Dashboard

A modern, responsive **Productivity Dashboard** built using **HTML, CSS, and JavaScript**. This single-page application combines multiple productivity tools into one intuitive interface, allowing users to manage tasks, plan their day, stay focused, and access useful information without leaving the dashboard.

## Features

* Dashboard Navigation
* Todo List
* Daily Planner
* Daily Goals Tracker
* Pomodoro Timer
* Motivation Quote Generator
* Weather Widget
* Live Date & Time
* Dynamic Background
* Light/Dark Theme Toggle
* Local Storage Support
* Responsive Design

## Tech Stack

* HTML5
* CSS3
* JavaScript (ES6)
* Fetch API
* Local Storage API
* Geolocation API
* Date Object
* setInterval()

## Features Overview

### Dashboard Navigation

Navigate between different productivity tools using an interactive dashboard without reloading the page.

### Todo List

* Add tasks
* Mark tasks as completed
* Mark important tasks
* Delete tasks
* Data saved using Local Storage

### Daily Planner

Plan your day by adding notes to hourly time slots.

### Daily Goals

Create daily goals, track progress, and monitor completed goals.

### Pomodoro Timer

A focus timer with Start, Pause, and Reset controls to improve productivity.

### Motivation Quotes

Fetch random motivational quotes using a public Quotes API.

### Weather Widget

Display real-time weather information using a Weather API and the user's location.

### Live Date & Time

Continuously updates the current date and time.

### Dynamic Background

Automatically changes the dashboard background based on the current time of day.

### Theme Switch

Toggle between Light Mode and Dark Mode with the selected theme stored in Local Storage.


Open `index.html` in your browser.
